Popular uses

Make an Urdu profile name, WhatsApp caption, gaming name, poetry heading, Eid greeting or social bio. For longer writing, use the basic Urdu editor or rich text editor.

Is this a translator?

No. Roman Urdu input is transliteration: it changes sounds into Urdu script. It does not translate English meaning. Direct mode keeps English and Urdu exactly as entered.

Are favourites uploaded?

No. Text, favourites and recent inputs are stored only in this browser. The optional Roman Urdu suggestion service is described in the privacy policy.
