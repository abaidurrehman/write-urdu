OUR PURPOSE

Why write Urdu online?

Urdu should be easy to create wherever your ideas begin. Write Urdu turns familiar Roman Urdu into readable Urdu, then gives you the tools to refine, save and share it.

Who maintains Write Urdu

Write Urdu is an independently maintained web project. Updates focus on browser compatibility, Urdu text rendering, privacy and practical writing tools. The project does not publish a personal maintainer name or home address.

For a correction or accessibility report, email admin@write-urdu.com through the feedback page. Please describe the page, browser and steps to reproduce the issue, and remove private text from reports.

Editorial and correction policy

Guides are reviewed against the current interface and tested in a modern browser before substantial updates. Reported errors are reproduced where possible, corrected in the source, and rechecked with the site's static and browser tests.

How the project is maintained

Write Urdu is an independent browser-based project that helps people create readable Urdu without installing specialist desktop software. The public feedback page is the correction channel for inaccurate suggestions, accessibility problems and outdated instructions.

  • Make Urdu writing more accessible to speakers, learners and diaspora communities.
  • Convert familiar Roman Urdu typing into right-to-left Urdu script.
  • Support everyday communication in messages, email, documents and social media.
  • Provide both a simple editor and a rich-text editor for formatted work.
  • Offer an on-screen Urdu keyboard for direct character entry.
  • Provide browser-based Card Studio and static QR tools for shareable Urdu designs.

Known limitations

  • Roman Urdu suggestions are transliteration hints, not a translation service, and ambiguous spellings can produce more than one possible Urdu word.
  • Word, PDF, PNG and sharing behavior depends on the browser and device.
  • QR payload handling can vary between scanner applications; review important content before sharing it.

Help us improve

Clear feedback helps us improve accuracy, accessibility and ease of use. If you find a problem or have a practical suggestion, please visit the Feedback page.
