Written steps for typing Urdu online

Use these instructions alongside the videos. They describe the current browser tools in plain language so you can follow the workflow without sound or an embedded player.

1. Convert Roman Urdu in the basic editor

  1. Open the basic editor and place the cursor in the writing area.
  2. Type a Roman Urdu word with English letters, then press Space.
  3. Review the Urdu suggestion and choose an alternative when one is shown.
  4. Use Copy text when you are ready to paste the result into another app.

Roman Urdu transliteration changes the script; it is not the same as translating the meaning into another language.

2. Format a longer document

  1. Open the Urdu Rich Text Editor for headings, lists, fonts, colour and alignment.
  2. Select the text you want to format and use the toolbar controls.
  3. Save a local draft or import plain text when you need to continue later.
  4. Export to Word, PDF or PNG after checking the preview.

3. Create a shareable Urdu image

From either editor, choose Create Urdu Card to open Card Studio with the selected text. Pick a template and output size, adjust the text box, and download the finished PNG. Uploaded backgrounds stay on your device.
