Copy, export, print and share Urdu

Write Urdu features and export options

Write Urdu converts Roman Urdu into Urdu script as you type. Once your text is ready, you can copy, share, print or download it in several useful formats. This guide explains each option.

1. Save a plain-text file

A text file is lightweight and easy to open in Notepad, TextEdit or another plain-text editor. Saving a copy also protects your work if you close or refresh the page.

  1. Finish typing your Urdu in the editor.
  2. Open Export and enter a filename.
  3. Select Text file to download a .txt copy.

2. Copy text to the clipboard

Use the clipboard when you want to paste your Urdu into an email, document, message or social media post without downloading a file.

  1. Select Copy text above the editor.
  2. Paste the text in another app with Ctrl+V, Command+V or your device's Paste command.

3. Print Urdu text

The print option opens your browser's print dialog, where you can choose a printer, page size and other available settings.

  1. Open Export and select Print.
  2. Review the preview, choose your print settings and confirm.

4. Export a Word document

Choose Word format when you want an editable document that can be opened in Microsoft Word or another compatible word processor.

  1. Open Export.
  2. Select Word document to download the file.

5. Export a PDF

PDF is useful when you want to preserve the document's appearance for reading, printing or sharing. Recipients can open it without having the same editor configuration.

  1. Open Export.
  2. Select PDF document and wait for the download to finish.

6. Export a PNG image

An image is convenient for presentations, visual posts and apps that do not handle Urdu text consistently.

  1. Open Export and select PNG image.
  2. Save or share the downloaded PNG using your device's normal controls.

7. Share with WhatsApp

The Share button prepares your Urdu text for WhatsApp. Depending on your device, WhatsApp may open in the installed app or in the browser.

  1. Select Share above the editor.
  2. Choose a conversation in WhatsApp and review the text before sending it.

8. Use the Urdu Rich Text Editor

Open the Urdu Rich Text Editor when you need headings, lists, colours, alignment, multiple fonts or other document formatting.

9. Recover local drafts

The editors save recent drafts in this browser only. No account or server upload is required, and the saved text stays on this device unless you clear it.

  1. Type or paste text into any Write Urdu editor.
  2. Open Recent drafts below the editor.
  3. Select a previous draft to restore it, or choose Clear history to remove local copies.

10. Import a text file

Use Import text to bring a UTF-8 plain-text file into the editor. Imported content is processed locally and can then be converted, formatted, copied or exported normally.
