# SEO and discoverability backlog

Technical SEO makes the tools understandable; it does not create external authority by itself. Keep this work genuine:

- keep official social profiles and product descriptions consistent;
- publish real tutorials and current screenshots on the maintained YouTube channel;
- share useful Urdu-typing guidance with education and language communities where appropriate;
- seek honest reviews and citations rather than paid link schemes;
- maintain a small changelog only when releases are actually shipped;
- never publish fake testimonials, ratings, usage numbers or authors.

Potential original guides are tracked separately from the technical SEO work so each page can include tested examples and a real review date.
